# Nikhilesh Gayakwad - Personal Portfolio

A premium, modern, and professional single-page developer portfolio designed to present a genuine, high-quality showcase for a B.Tech Computer Science Engineering student. Optimized for internship applications, LinkedIn profiles, college presentations, and placement preparation.

🚀 **Live Demo: https://portfolio-green-ten-87.vercel.app/
---

## 🎨 Visual Highlights & Aesthetics

Inspired by 2026 startup-style landing pages, this portfolio features high-end developer-themed design tokens:
*   **Premium Dark Theme:** Rich slate background tones paired with subtle indigo, purple, and teal gradients.
*   **Animated Gradient Background Blobs:** Floating blurred gradient spheres that drift gracefully across the screen using smooth, hardware-accelerated CSS keyframes.
*   **Interactive 3D Profile Card:** A custom visual widget featuring floating glass pill badges (Java, DSA, Web Dev) that responds with a smooth 3D perspective tilt on mouse hover.
*   **Semantic Timeline Architecture:** An elegant timeline showing Class X, Class XII, and B.Tech CSE engineering scores (CGPA: 7.90) at Bansal College of Engineering.
*   **Graphical DSA Progress Tracker:** Visual dashboard cards showing completed algorithms topics with checkmarks and progress bars, as well as ongoing learning (Graphs & DP) with pulsing active nodes.
*   **Responsive Mobile-First Design:** Fully fluid layouts utilizing CSS Flexbox and Grid, with a customized menu panel built for mobile viewports.

## 🛠️ Technology Stack

This project is built from scratch without bulky framework dependencies, ensuring sub-second load times, excellent lighthouse audits, and ease of customization:
*   **Markup:** Semantic HTML5 (SEO friendly, structured meta elements, embedded high-fidelity vector inline SVGs)
*   **Styling:** Modern CSS3 (Custom variables, glassmorphic backing filters, keyframe animations, responsive grids)
*   **Logic:** Vanilla ES6 JavaScript (IntersectionObserver scroll spying, mobile menu triggers, quick copy-to-clipboard utilities)

## ⚙️ How to Customize Locally

### 1. Change Profile Photo
Save your photo in `assets/images/` and rename it to exactly `profile.jpg` (replace the current placeholder). The CSS is preconfigured with `object-fit: cover` and `object-position: center top` to perfectly center your face.

### 2. Update Resume File
Replace the template `resume.pdf` inside the `assets/` folder with your own PDF resume.

### 3. Edit Personal Content
Open `index.html` and search for any text blocks (like CGPA, phone number, projects text) to adjust them as needed.
